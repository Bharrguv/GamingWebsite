    About game

    Minecraft is develop by Mojang 4J Studios. In this game player can materialize his imagination. By building blocks over block. This game is so addictive that often you forget to eat & sleep. There is freedom for the kids to harvest there imaginations and stylize the game as they have wanted. Minecraft game lacks any specific purpose or mission. You just have to play the game as you want to play. It has different game modes including arcade and survival. If you like game which is only arcade mode then download and install Adventure Island, Crazy Birds. These are only arcade game. But minecraft support survival mode too. In survival mode the experience of a player comes out to handle the given situations.
